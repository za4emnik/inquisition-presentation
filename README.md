# README

### Clone repo:
```
git clone https://github.com/za4emnik/inquisition-presentation.git
cd inquisition-presentation
rails db:migrate
```

### Add gem to Gemfile:
```
gem 'inquisition', :git => 'git://github.com/rubygarage/inquisition.git', :branch => "feature/166-create-additional-elements"
bundle install
```

### Run commands:
```
inquisition build
inquisition setup
inquisition analyze
```
